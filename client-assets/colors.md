# Brand Colors — A Scuba Guide

**Status:** Not yet chosen. Phase 4 (`design-assist`) responsibility. Tyler will direct.

## Constraints decided so far

- **Do NOT inherit** the existing PWA's palette (`#082f49` ocean, `#0c4a6e` deep, `#075985`, `#0369a1`, `#0ea5e9` cyan, `#f97316` coral). Confirmed with Tyler in routing — fresh identity.
- **No purple/blue gradients by default** (Stocks Local rule #5).
- **Must work for a 3D-immersive scroll-animated experience** — likely benefits from a darker base palette so 3D elements pop. Final call in Phase 4.

## Banned

- The exact ocean+coral palette from the existing PWA (would make the marketing site feel like the same product, not a brand layer above it).
- Default Tailwind `slate-900` / `cyan-500` combo (too generic — every dive site on the internet uses it).
- Inter / Space Grotesk default fonts (Stocks Local rule #5 — Tyler picks fonts in Phase 4).
